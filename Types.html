My Journey

Normal Type



- Normal Type hlavně obsahuje v podstatě normální zvířata, jako ptáci, savci a Bůh  
- V současné době je 133 Normal Type Pokémonů
- Nejsou silní proti žádnému typu
- Mají slabinu proti Figting Type


Fighting Type



- Figting Type hlavně obsahuje Pokémony schopných bojového umění  
- V současné době je 86 Fighting Type Pokémonů
- Jsou silní proti Normal, Rock, Ice, Steel, Dark Type
- Mají slabinu proti Flying, Psychic, Fairy Type


Flying Type



- Flying Type hlavně obsahuje Pokémony schopných letu  
- V současné době je 115 Flying Type Pokémonů
- Jsou silní proti Fighting, Bug, Grass Type
- Mají slabinu proti Electric, Rock, Ice Type


Poison Type



- Poison Type hlavně obsahuje jedovaté Pokémony, nebo Pokémony se schpností otrávit
- V současné době je 89 Poison Type Pokémonů
- Jsou silní proti Fairy, Grass Type
- Mají slabinu proti Ground, Psychic Type


Ground Type



- Ground Type hlavně obsahuje Pokémony nějak spojené se zemí, hlínou atd.
- V současné době je 79 Ground Type Pokémonů
- Jsou silní proti Electric, Fire, Poison, Rock Type
- Mají slabinu proti Grass, Ice, Water Type


Rock Type



- Ground Type hlavně obsahuje Pokémony nějak spojené s kameny, vetšínou jejich samotné tělo, nebo část je vytvořené z kamenů
- V současné době je 80 Rock Type Pokémonů
- Jsou silní proti Bug, Fire, Flying, Ice Type
- Mají slabinu proti Grass, Fighting, Water, Steel, Ground Type


Bug Type



- Bug Type hlavně obsahuje Pokémony, kteří jsou brouci, nebo i korýši 
- V současné době je 94 Bug Type Pokémonů
- Jsou silní proti Dark, Grass, Psychic Type
- Mají slabinu proti Fire, Flying, Rock Type


Ghost Type



- Ghost Type hlavně obsahuje Pokémony, kteří jsou duchové 
- V současné době je 76 Ghost Type Pokémonů
- Jsou silní proti Ghost, Psychic Type
- Mají slabinu proti Ghost, Dark Type


Grass Type

- V současné době je 132 Grass Type Pokémonů
- Jsou silní proti Water, Rock, Ground Type
- Mají slabinu proti Bug, Fire, Flying, Ice, Poison Type
